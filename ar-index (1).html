The Spiral Project — AR Setup

Step 1 — Print the target Open target.html in your browser and print it.
Minimum size: 10cm × 10cm. Flat surface, good lighting.
Step 2 — Compile your .mind file (60 seconds) Go to:
hiukim.github.io/mind-ar-js-doc/tools/compile

Upload a photo or screenshot of the printed target.
Click Compile → download targets.mind
Step 3 — Add to repo & push Put targets.mind in the same folder as this file.
Push to GitHub. Enable GitHub Pages.
Open the Pages URL on your phone → point at target.
This screen disappears automatically once targets.mind is detected.
HTTPS is required — GitHub Pages provides this.
